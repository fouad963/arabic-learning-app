# Arabic Learning App

This repository contains the source code for an AI-powered app to help students learn Arabic with personalized learning paths.

## Project Structure

- `frontend/`: Contains the React Native code for the mobile application.
- `backend/`: Contains the Node.js/Express server code.
- `database/`: Contains the database schema and related scripts.
- `ml_models/`: Contains the machine learning models for speech recognition and NLP.

## Technologies Used

- React Native
- Node.js with Express
- MongoDB
- TensorFlow, PyTorch
- Google Speech-to-Text, Microsoft Azure Cognitive Services
- spaCy, NLTK
